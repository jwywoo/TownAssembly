## Town Assembly
### Town Assembly는 정치인과 예비 정치인들을 위한 홍보 플랫폼입니다.
> 자신의 활동을 직접 올리고 자신의 의견을 오늘의 한마디로 업로드해서 자신을 홍보하고 유권자들과 소통할 수 있는 플랫폼입니다.
